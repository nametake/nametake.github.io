GCPの認証済みのhttp.Clientの取得方法

November 9, 2018 1-minute read
GCP • Go

GCPの認証済みのhttp.Clientが欲しいときにちょっとだけ調べたのでメモ。

結論だけ言うと、golang.org/x/oauth2/googleのDefaultClientを使えばいいみたいです。

今回はDataflowのAPIを叩きたかったので以下のようにすればできました。

client, err := google.DefaultClient(ctx, dataflow.CloudPlatformScope)
if err != nil {
	return fmt.Errorf("failed create client: %v", err)
}
